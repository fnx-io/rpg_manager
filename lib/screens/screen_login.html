This is something between technology preview and hobby project. It's a RPG manager - hire heroes, train them, send them to quests. Pure RPG without actual walking! Yay!

RPG manager was build with:

  • Dart
  • Angular 2
  • fnx_ui - set of pure Dart Angular 2 components for form-heavy desktop applications
  • Firebase - unofficial Dart wrapper for Firebase
  • game-icons.net - enormous set of fantasy SVG icons
  • Material icons
